# JFXS and how to use it

## how to compile & run

```
1) download java
2) download javafx
3) javac --module-path %PATH_TO_FX% --add-modules javafx.controls Main.java
4) java --module-path %PATH_TO_FX% --add-modules javafx.controls Main
```
